Welcome to E-FileTypes-Py's documentation!
===================================

**E-FileTypes-Py** is a Python library for that securely encrypts files of any type. Can be used to encrypt files for secure storage on your computer or sent over the Internet as well as storage on cloud servers. The encrypted file can only be decrypted with the correct password.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
